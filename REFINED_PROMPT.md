# Refined Production Prompt

Design, develop, and deploy a production-ready conference website and integrated
application portal for **AfriQA 2026: Quantum Science & Technology Across
Africa**, hosted at the AIMS Research Innovation Centre in Kigali, Rwanda from
30 November to 4 December 2026.

The website must feel like a flagship scientific conference experience:
premium, academic, visually striking, accessible, mobile-first, and fast. The
design language should be quantum-anchored rather than generic futuristic tech:
draw from probability amplitudes, wave interference, Bloch spheres, entanglement
graphs, quantum circuits, crystal lattices, and rigorous scientific
visualisation.

Use the AfriQA logo and AIMS Research and Innovation logo as the only marks in
the header brand area. Do not place typed text such as "AfriQA 2026" beside the
logos in the header. Use the attached schedule spreadsheet as the primary
programme source and preserve the scientific integrity of the event. Reimagine
the existing public AfriQA event content; do not copy the current website.

Build the public website sections:

- Home
- About the Conference
- Scientific Themes
- Programme Theme Map
- Programme
- Keynote Speakers and Contributors
- Organising Committee
- Registration
- Applications
- Abstract Submission
- Scholarships and Travel Support
- Venue
- Accommodation
- Travel Information
- Sponsors and Partners
- News and Announcements
- FAQs
- Contact

The application portal is the centrepiece. It must be production-only, with no
mock/demo data and no fake submissions. Implement the backend using Google Apps
Script, Google Sheets as the database, and Google Drive for uploads.

Portal requirements:

- User registration and authentication
- Applicant dashboard
- Conference registration
- Sponsorship category: fully sponsored, partially sponsored, self sponsored
- Travel grant/scholarship application workflow
- Abstract submission
- File uploads for CV, passport/ID, abstract PDFs, letters, and supporting docs
- Save-and-continue through the backend
- Email confirmations and notifications
- Application status tracking
- Admin review portal
- Searchable applicant database
- Status updates
- CSV export
- Server-side validation and admin email allowlist

Deployment requirements:

- Static HTML/CSS/JavaScript website deployed through GitHub Pages
- GitHub Actions workflow for Pages deployment
- Google Apps Script Web App backend
- Google Sheets database with tabs for users, sessions, applications, files, and audit logs
- Google Drive upload folder for applicant documents
- Step-by-step production setup guide

Avoid generic neon AI visuals, stock technology motifs, decorative bokeh, and
science-fiction clichés. Prioritise scientific clarity, polished micro-
interactions, generous whitespace, strong typography, and a visual narrative
that Africa is actively shaping the future of quantum science.
