About the conference

A rigorous conference experience built around Africa's quantum capability.

AfriQA 2026 convenes leading researchers, academics, emerging scientists, and partners across the African continent to strengthen collaboration and advance quantum research. The conference is hosted at the AIMS Research Innovation Centre in Kigali and includes a summer school, tutorials, research highlights, poster sessions, project breakouts, and stakeholder engagement.

The visual language of this website is anchored in scientific concepts: probability amplitudes, superposed states, entanglement graphs, wave interference, lattice order, and Bloch-sphere geometry. The result is intentionally academic, precise, and usable.

        FAQs

        Quick answers for applicants and delegates.

        Can I save my application and return later?

        Yes. Drafts are saved through the secure application system once registration is open.

        How are files handled?

        The Apps Script backend stores uploaded files in Google Drive and writes document metadata to the Google Sheet database.

        Is travel support available?

        The application flow supports fully sponsored, partially sponsored, and self sponsored delegates. Organisers can review the stated sponsorship category and requested amount in the admin portal.

        Who can access the admin portal?

        The Apps Script backend includes admin email validation. Only approved admin accounts should be able to review, export, or update statuses.
